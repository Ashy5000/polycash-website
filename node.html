Run a Node

You can run a full node on the Polycash testnet, getting rewarded for your contribution to the network's security and decentralization. Nodes can run on relatively modest hardware, and do not consume large amounts of energy, thanks to the APoW algorithm.

To get started, follow the steps below.

  1. Download the reference client from here.
  2. Install prerequisites listed in the Github's README.
  3. Build the node executable:
    go build -o builds/node/node
  4. Move into the contracts directory and build the VM executable:
    cd contracts; cargo build; cd ..
  5. Create a key file:
    touch key.json
  6. Create a key:
    ./builds/node/node -command "keygen"
  7. Bootstrap peers: ./builds/node/node -command "bootstrap"
  8. Start your node: ./builds/node/node_linux-amd64 -serve -mine -port 8080
  9. Wait for rewards!
